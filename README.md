# CO₂ Emissions Visualisation System (Malaysia & ASEAN)

### Project Overview
This project visualises per-capita CO₂ emissions for Malaysia, ASEAN countries, and the world.
Developed for BAI21124 Data Processing & Visualisation (Sept 2024).

### System Structure
- `notebooks/` – data cleaning and preprocessing
- `data/` – cleaned datasets
- `app.py` – Streamlit web application
- `requirements.txt` – dependencies

### How to Run
```bash
pip install -r requirements.txt
pip install plotly streamlit-folium folium
streamlit run app.py
